# Volleyball Trainer (v2)

A next-generation Progressive Web App for learning volleyball rules, rotations, and strategy — the successor to [volleyball-trainer](https://github.com/inspectorgad/volleyball-trainer).

## Status

This repo is the planned v2 of the Volleyball Trainer app. Development in progress.

## Predecessor

The original [volleyball-trainer](https://github.com/inspectorgad/volleyball-trainer) includes:

- **Simulator Mode** — Interactive court with 5-1, 6-2, and 4-2 offensive systems
- **Play-Along Mode** — 25 realistic rally scenarios with rotation tracking
- **Rules Hub** — 20 rules, progress tracking, 30-question quiz bank
- **NCAA 2025 Tournament** — Live bracket tracking with Kansas Jayhawks path highlights

## Tech Stack (planned)

- Progressive Web App (PWA)
- Installable on iOS, Android, and desktop
